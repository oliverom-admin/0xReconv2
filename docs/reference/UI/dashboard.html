Credential Vaults

Manage credential vaults and secret stores for scan operations

🔐

Centralized Secret Management

CAIP securely manages credentials for scanning external systems. Instead of storing credentials directly, you register connections to external secret stores (Azure Key Vault, HashiCorp Vault, AWS Secrets Manager, or encrypted local files).

How it works:

  • Register Stores: Connect to your secret management systems by providing connection details
  • View Secrets: See available secrets in each store (metadata only, values are never displayed)
  • Reference in Scans: Use secret references in scan configurations instead of plaintext credentials
  • Runtime Resolution: CAIP resolves actual credentials at scan time directly from your trusted secret store

Security Benefits: Credentials are never stored in CAIP, never logged, and always retrieved fresh from your authoritative source.
